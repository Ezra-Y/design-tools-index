# 色彩工具（配色 + 渐变）

配色与渐变生成工具的完整列表（精选见 `SKILL.md`）。条目来自 [Awesome-Design-Tools](https://github.com/goabstract/Awesome-Design-Tools)。

## 配色
- [Coolors](https://coolors.co/) — the super-fast color scheme generator.
- [Adobe Color](https://color.adobe.com/explore/) — create schemes with the color wheel or browse community combinations.
- [Color Hunt](https://colorhunt.co/) — thousands of hand-picked, trendy color palettes for inspiration.
- [Khroma](http://khroma.co/) — AI-based tool to generate color palettes from your preferences.
- [Paletton](https://paletton.com) — a designer tool for creating color combinations that work together well.
- [Color Hexa](https://www.colorhexa.com/) — info on any color plus matching palettes.
- [Sip](https://sipapp.io/) — global eyedropper and color organizer for macOS.
- [Picular](https://picular.co/) — a rocket-fast primary color generator using Google image search.
- [Pigment](https://pigment.shapefactory.co/) — a color palette generator with many configurations.
- [ColorKit](https://colorkit.io) — blend colors and generate a color's shades and tints.

## 渐变
- [UI Gradients](https://uigradients.com/) — a handpicked collection of beautiful color gradients.
- [CSS Gradient](https://cssgradient.io/) — a free CSS gradient generator for colorful backgrounds.
- [Web Gradients](https://webgradients.com/) — 180 free linear gradients usable as content backdrops.
- [Grabient](https://www.grabient.com/) — a beautiful, simple UI for generating web gradients.
- [Gradient Hunt](https://gradienthunt.com/) — thousands of trendy hand-made gradient palettes.
- [CoolHue](https://webkul.github.io/coolhue/) — handpicked gradient palettes and swatches.
- [Gradienta](https://gradienta.io/) — multicolor CSS gradients with JPG downloads.
- [ColorSpace](https://mycolor.space/) — generate matching color schemes and palettes.
