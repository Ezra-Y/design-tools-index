# 交互与动效（原型 + 动画）

原型与动效工具的完整列表（精选见 `SKILL.md`）。条目来自 [Awesome-Design-Tools](https://github.com/goabstract/Awesome-Design-Tools)。

## 动画工具
- [Lottie](https://airbnb.io/lottie/) — parses AE animations exported as JSON and renders them natively on mobile and web.
- [GSAP](https://greensock.com/) — a suite of tools for scripted, high-performance HTML5 animations in all major browsers.
- [After Effects](https://www.adobe.com/products/aftereffects.html) — Adobe's motion graphics, visual effects and compositing app.
- [Haiku Animator](https://www.haikuforteams.com/) — keyframe-based animation connecting UI tools to code developers can use.
- [Flare](https://www.2dimensions.com/about-flare) — design and animation tool for app or game animations.
- [Kite Compositor](https://kiteapp.co/) — a powerful animation and prototyping app for Mac & iOS.
- [Keyshape](https://www.keyshapeapp.com/) — 2D animation tool with animated SVG export.
- [Tumult Hype](https://tumult.com/hype/) — HTML5 animation/interactives creation app for macOS.
- [Spirit](https://spiritapp.io/) — create and manage animations real-time in the browser.

## 原型
- [Framer](https://framer.com/) — visually design realistic interactive prototypes.
- [InVision](https://www.invisionapp.com/) — prototyping, collaboration and workflow platform.
- [ProtoPie](https://www.protopie.io/) — piece hi-fi interactions and sensor-aided prototypes together in minutes.
- [Marvel](https://marvelapp.com/) — wireframe, prototype, design and create specs in one place.
- [Origami](https://origami.design/) — a free tool for designing modern UIs; run prototypes on iPhone/iPad.
- [Principle](https://principleformac.com/) — design animated and interactive user interfaces.
- [Axure RP](https://www.axure.com/) — wireframing, prototyping, collaboration and specifications.
- [UXPin](https://www.uxpin.com/) — build prototypes that feel real with code components, logic and states.
- [Proto.io](https://proto.io/) — create fully-interactive high-fidelity prototypes.
- [Maze](https://maze.design/) — analytical KPIs for your InVision prototypes.
