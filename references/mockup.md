# 样机（Mockup）

样机与设备框展示工具的完整列表（精选见 `SKILL.md`）。条目来自 [Awesome-Design-Tools](https://github.com/goabstract/Awesome-Design-Tools)。

## Mockup
- [Smartmockups](https://smartmockups.com/) — create product mockups in just a few clicks.
- [Artboard Studio](https://artboard.studio/) — online graphic design app focused on product mockups.
- [Device Shots](https://deviceshots.com) — create device mockups from screenshots, for free.
- [Mockup World](https://www.mockupworld.co/) — free, layered, customizable photo-realistic PSDs.
- [Rotato](https://www.rotato.xyz/) — animated 3D mockups for app designs.
- [Mockuuups](https://mockuuups.studio/) — drag-and-drop tool for app preview mockups.
- [Screely](https://www.screely.com/) — quickly frame web designs in a minimalist browser window.
- [Threed](http://threed.io/) — generate custom 3D device mockups in your browser.
- [Mockflow](https://www.mockflow.com/) — the quickest way to brainstorm UI ideas.
