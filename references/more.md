# 更多设计工具类目 (More Design Tool Categories)

本文件收录 `SKILL.md` 未展开的其余设计工具类目，以及高频类目中未列入的补充条目。每条为外部链接 + 一句话说明。完整原始列表见上游 README：https://github.com/goabstract/Awesome-Design-Tools

## 无障碍工具 (Accessibility)
- [Axe](https://www.deque.com/axe/) — accessibility engine for automated testing across browsers and frameworks.
- [WAVE](https://wave.webaim.org/) — evaluate web content for accessibility issues in Chrome and Firefox.
- [Stark](https://getstark.co/) — simulate color blindness and check contrast inside Sketch.
- [Color Oracle](https://colororacle.org/) — a free color blindness simulator.
- [Leonardo](https://leonardocolor.io) — generate color palettes by desired WCAG contrast ratio (Adobe).
- [Contrast](https://usecontrast.com/) — macOS app for quick WCAG color contrast ratios.
- [PA11Y](http://pa11y.org/) — command-line accessibility testing for web pages.

## 增强现实 (Augmented Reality)
- [Spark AR Studio](https://www.sparkar.com) — create AR experiences for Instagram without code.
- [Lens Studio](https://lensstudio.snapchat.com/) — build and publish AR for Snapchat.
- [Unity](https://unity.com/) — build 3D/2D games and deploy across mobile, desktop and VR/AR.
- [Vectary](https://www.vectary.com/) — create 3D and AR for your website.
- [Torch](https://www.torch.app/) — cloud-based 3D design and prototyping for mobile AR.
- [Lightform](https://lightform.com/) — a design tool for projected AR.
