# 设计交付与转码（Design Handoff + Design to Code）

设计交付、标注 Spec 与设计转代码工具的完整列表（精选见 `SKILL.md`）。条目来自 [Awesome-Design-Tools](https://github.com/goabstract/Awesome-Design-Tools)。

## 设计交付 (Design Handoff)
- [Zeplin](https://zeplin.io/) — handoff designs and style guides with accurate specs, assets and code snippets.
- [Avocode](https://avocode.com) — open designs without design tools; click layers to get code.
- [InVision Inspect](https://www.invisionapp.com/feature/inspect/) — prepare designs for development.
- [Sympli](https://sympli.io) — automated specs and assets handoff, integrated with Jira, Xcode and Android Studio.
- [Sketch Measure](https://github.com/utom/sketch-measure) — Sketch redline plugin annotating distance and size of elements.
- [Flawless App](https://flawlessapp.io/) — compare the expected design vs. the real app inside iOS Simulator.
- [Specctr](https://specctr.com) — creates redline annotations for designs in PS, AI, or ID.

## Design to Code
- [Webflow](https://webflow.com/) — build responsive sites in the browser, then host or export the code.
- [Anima](https://www.animaapp.com/) — a Sketch plugin/web app that converts Sketch designs to HTML.
- [Supernova Studio](https://supernova.io/) — convert Sketch designs into Android, iOS or React Native code.
- [PaintCode](https://www.paintcodeapp.com) — instantly convert vector drawings into Swift, Obj-C, JS or Java code.
- [Pinegrow](https://pinegrow.com/) — visual editor for CSS Grid, Bootstrap, Foundation and HTML/CSS.
- [Bootstrap Studio](https://bootstrapstudio.io/) — create responsive websites with Bootstrap.
- [Draftbit](https://draftbit.com/) — visually design and build mobile apps from the browser.
- [Tilda](https://tilda.cc/) — build sites, landing pages and stores with modular blocks.
- [Blocs](https://blocsapp.com/) — visual responsive web design tool without code.
